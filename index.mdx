# Getting Set Up:

- Before getting started, download the relevant files and repos:

## LeRobot Database

Download the Source Code (One-Time):

```bash
git clone https://github.com/huggingface/lerobot.git
cd lerobot
```

Set Up Conda Environment (One-Time)

```bash
conda create -y -n lerobot python=3.10
conda activate lerobot

# if running MiniConda, also run;
conda install ffmpeg -c conda-forge
```

Install the LeRobot Packages;

```
pip install -e .
```

## Additional Extensions

### Hugging Face:

> Purpose: Necessary to store datasets and policies, make an account [here](https://huggingface.co/).

```bash
pip install --upgrade huggingface_hub

huggingface-cli login # will be prompted to paste token after running this
```

### Weights and Bases

> Purpose: Plots relevant data to assist in training policies. Make an account [<u>here</u>](https://wandb.ai/home) and log in to your terminal.

```bash
pip install wandb

# Make a API Key: https://wandb.ai/authorize

wandb login # paste key when prompted
```
